Custom User Stylesheets for Twitter

After receiving access to the new Twitter interface, you might have been left thinking “I’m not keen on all the clutter down the right hand side”. Well there is a way to get rid of it. Look at the before and after shots below:

Just follow these simple steps:

Step 1

Make sure you have Safari 5 and you have Safari Extensions enabled.

Step 2

Download and install the User CSS Manager . This will enable you to have Custom CSS for your favourite sites.

Step 3

Once installed (you may have to restart Safari) you should have a new icon in your Safari Toolbar (the icon is a Capital A). Click on the icon to open the User CSS Manager and then click on New User CSS.

**Settings I use for Twitter **

Name: Twitter

Domains: http://.twitter.com/

Styles: 

.trends-inner,.trends-inner+hr,.wtf-inner,.wtf-inner+hr,.definition,.definition+hr{display:none !important;}

Style references came from @shauninman

Anything I’ve missed or some style improvements just let me know.
